개인정보처리방침

시행일: 2026-03-09

글(이하 "앱")은 사용자의 개인정보를 소중히 여깁니다. 본 방침은 수집하는 정보, 사용 방법, 데이터 관리 방법을 설명합니다.

1. 수집하는 정보

  • 기기 ID (UUID) — 기기별로 생성되는 고유 식별자이며, 사용자 신원과 연결되지 않습니다.
  • 사용자 ID — 계정 및 사용량 관리를 위해 서버에서 생성되는 식별자입니다.
  • AI 사용량 — 사용량 제한을 위한 일일/월간 요청 수입니다.
  • 구독 상태 — 적절한 서비스 등급을 제공하기 위해 사용됩니다.
  • iCloud 동기화 데이터 — 동기화를 활성화하면 노트가 Apple iCloud Documents에 저장됩니다. 당사는 이 데이터에 접근하지 않습니다.

2. AI 기능

AI 기능을 사용하면 노트 내용이 처리를 위해 Cloudflare Workers 서버로 전송됩니다.

전송된 내용은 Anthropic의 Claude API로 처리된 후 즉시 폐기됩니다. 당사 서버에 노트 내용을 저장하지 않습니다.

AI 기능은 완전히 선택 사항입니다. 사용하지 않으면 노트 내용이 어떤 서버로도 전송되지 않습니다.

3. 데이터 저장

  • 노트 — 기기에 로컬 저장 + 선택적으로 iCloud Documents에 저장됩니다.
  • 인증 토큰 — 기기 Keychain에만 안전하게 저장됩니다.
  • 서버 — device_id, user_id, 사용량 데이터만 Cloudflare KV에 저장됩니다.

4. 데이터 삭제

  • 설정 > 계정 삭제 — 서버의 모든 데이터가 완전히 삭제됩니다.
  • 앱을 삭제하면 기기의 모든 로컬 데이터가 삭제됩니다.
  • iCloud 데이터는 기기의 Apple iCloud 설정에서 관리할 수 있습니다.

5. 제3자 서비스

  • Anthropic (Claude API) — AI 텍스트 처리. 내용은 저장되지 않습니다.
  • Apple (iCloud, StoreKit) — 클라우드 동기화 및 인앱 결제.
  • Cloudflare (Workers, KV) — API 서버 및 사용량 데이터 저장.

6. 문의

개인정보 관련 문의는 support@geul.app으로 연락해 주세요.
